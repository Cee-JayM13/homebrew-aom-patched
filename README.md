# Cee-jaym13 Aom-patched

## How do I install these formulae?

`brew install cee-jaym13/aom-patched/<formula>`

Or `brew tap cee-jaym13/aom-patched` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cee-jaym13/aom-patched"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
